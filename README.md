# yggdrash_poc
yggdrash proof of concept

```bash
    
    # brew install
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    
    brew install nodejs
    brew install npm
    brew install bower
    brew install nodemon
    
    
    npm install
    bower install
    
    # start yggdrash poc
    nodemon bin/www
    

```
